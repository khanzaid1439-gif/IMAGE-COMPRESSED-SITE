# 🖼️ Image Compressor Web App

A modern web app to compress images directly in the browser without uploading to any server.

## 🚀 Features
- Drag & Drop Upload
- Image Preview
- Compression Quality Control
- File Size Comparison
- Download Compressed Image

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Compressor.js

## 📸 How to Use
1. Upload or drag an image
2. Adjust quality
3. Click Compress
4. Download image
